# IA Score de Crédito

## 📌 Sobre
Este projeto foi desenvolvido com o objetivo de aplicar conceitos de análise de dados e machine learning para classificação de score de crédito.

## 💻 Tecnologias
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## 🎯 Objetivo
Criar um modelo simples capaz de classificar ou prever o score de crédito com base em dados disponíveis.

## 🚀 O que foi feito
- Tratamento e preparação dos dados
- Análise exploratória
- Aplicação de modelo de machine learning
- Avaliação básica dos resultados

## 📊 Aprendizados
- Fundamentos de machine learning
- Preparação de dados para modelos
- Análise de dados aplicada
- Interpretação de resultados
